# MishaNya's Portfolio

This repository contains the source code for my personal portfolio website, hosted on GitHub Pages. It showcases my skills, featured projects, and freelance experience as a Software Engineer.

**Live Site:** [mishanyacopilot.github.io](https://mishanyacopilot.github.io)

---

## About This Portfolio

This is a modern, responsive, single-page portfolio designed to provide a comprehensive overview of my technical expertise. It features dynamic project loading from JSON data and is styled with a clean, dark-mode aesthetic.

The portfolio is adapted from the original work of Viren Hirpara and has been significantly customized to reflect my personal projects and skills.

## Tech Stack

### Portfolio Website
- **HTML5**
- **CSS3** (with custom properties for theming)
- **JavaScript (ES6)** (for dynamic content rendering)
- **BrowserSync** (for local development)
- **npm** (for package management)

### Key Skills & Technologies Showcased
- **Backend:** Python (FastAPI), Go, Microservices, PostgreSQL, SQLite, Redis, RabbitMQ, Docker
- **AI/ML:** LLMs (Ollama), RAG, Whisper AI, Vector Databases (Qdrant)
- **Mobile:** Kotlin, Jetpack Compose, MVVM
- **Frontend:** React, JavaScript/TypeScript, HTML/CSS
- **DevOps & Monitoring:** Prometheus, Grafana, Zipkin
